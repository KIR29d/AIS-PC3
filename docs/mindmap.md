# Mind Map: АИС для магазина кастомных ПК

## Центр: CustomPC Manager (АИС)

- Администратор

  - Управление ролями
  - Просмотр аудита
  - Управление складами
- Менеджер
  - Работа с клиентами
  - Оформление заказов
  - Контроль сборки
  - Отчеты
- Сборщик
  - Получение заданий
  - Сборка ПК
  - Учет компонентов
  - Отчеты о сборке
- Клиент
  - Оформление заказов
  - Просмотр статусов
  - Личный кабинет (web)

### 5. Требования
- Функциональные
  - RBAC (Role-Based Access Control)
  - Валидация данных
  - Целостность данных
  - Аудит изменений
  - Многоскладской учет
- Нефункциональные
  - Производительность
    - < 2 сек отклик
    - 50 одновременных пользователей
  - Надежность
    - 99% доступность
    - Backup ежедневно
  - Безопасность
    - Хэширование паролей
    - HTTPS
    - Защита от SQL injection
  - Масштабируемость
    - Горизонтальное масштабирование
    - Микросервисы (перспектива)
