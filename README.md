# АИС CustomPC Manager

Автоматизированная информационная система для магазина кастомных персональных компьютеров.

## 📋 Описание проекта

Система предназначена для автоматизации бизнес-процессов магазина кастомных ПК:
- Управление заказами клиентов
- Сборка компьютеров по индивидуальным конфигурациям
- Складской учет компонентов
- Контроль работы сотрудников
- Веб-интерфейс для покупателей (в перспективе)

## 👥 Роли пользователей

| Роль | Функции |
|------|---------|
| **Администратор** | Управление пользователями, ролями, складами, просмотр аудита |
| **Менеджер** | Работа с клиентами, оформление заказов, контроль сборки |
| **Сборщик** | Получение заданий, сборка ПК, учет компонентов |
| **Клиент** | Оформление заказов, просмотр статусов (web) |

## 🔄 Статусы

### Заказы
`новый` → `в_работе` → `собран` → `отгружен` → `выполнен`

### Сборочные задания
`ожидает` → `в_сборке` → `готово`
