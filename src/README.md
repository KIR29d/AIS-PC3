# CustomPC Manager - АИС для магазина кастомных ПК

## Структура проекта

```
src/
├── CustomPC.Core/          # Ядро системы (Entities, Interfaces)
│   ├── Entities/           # Сущности базы данных
│   ├── Interfaces/         # Интерфейсы репозиториев и сервисов
│   ├── Enums/              # Перечисления
│   └── Services/           # Реализации сервисов (будущее)
├── CustomPC.Infrastructure/# Инфраструктура (DAL, EF Core)
├── CustomPC.API/           # Web API (ASP.NET Core)
└── CustomPC.WPF/           # Desktop приложение (WPF)
tests/                      # Тесты
```

## Описание

АИС CustomPC Manager предназначена для автоматизации бизнес-процессов магазина кастомных персональных компьютеров:

- Управление заказами клиентов
- Управление сборкой ПК
- Складской учет компонентов
- Управление сотрудниками и отделами
- Аудит изменений в системе

## Технологии

- **.NET 8** - платформа разработки
- **C#** - язык программирования
- **Entity Framework Core** - ORM для работы с данными
- **MS SQL Server** - система управления базами данных
- **WPF** - desktop интерфейс
- **ASP.NET Core** - web API (перспектива)

## Статусы в системе

### Заказы
- новый
- в_работе
- собран
- отгружен
- выполнен

### Сборочные задания
- ожидает
- в_сборке
- готово

## Роли пользователей

- Администратор - полный доступ к системе
- Менеджер - работа с клиентами и заказами
- Сборщик - выполнение сборочных заданий
- Клиент - просмотр своих заказов (перспектива)
